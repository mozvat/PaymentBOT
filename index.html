RaspberryPI and Payments

Download as .zip Download as .tar.gz View on GitHub

This is a fun research and development project developing concepts around hardware and software solutions within the payments technology industry. I decided to make a mini-POS that accepts encrypted magnetic data with a $35 chip set, RaspberryPI http://www.raspberrypi.org/ using a flavor of Linux, Debian. Then requests an authorization-for-payment from processors with a 'twist'. This project has been the most exciting personal R&D projects!! Mixing newer but older technologies has been a blast!!
